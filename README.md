# Simple fantasy game written in **BASH** 

I used **bash** to create a simple fantasy game, for now there are few classes you can choose from including mage, knight and archer, all of them have different stats and advantages.

## Classes
- knight does the most damage
- archer has the highest speed
- and mage deals magic + physical damage

## More about
It's very simple game with only one enemy for now. I added a feature of speed which decreases when you use block or heavy attack. It will be used to add a delay between enemies fights.

## Status
This project is a **work in progress** — more features are planned.

## How to run (Linux and macOS only)
To run this code you need to run these commands in your terminal:
chmod +x fantasygame.sh
./fantasygame.sh
- You need to run these commands in **directory where the game is**, should be ~/Downloads/bash_game/ depends on your language.
