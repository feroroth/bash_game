# Simple fantasy game written in **BASH** 

I used **bash** to create a simple fantasy game, for now there are few classes you can choose from including mage, knight and archer, all of them have different stats and advantages.

## Classes
- knight does the most damage
- archer has the highest speed and can attack before each fight
- and mage deals magic + physical damage

## More about
It's a very simple game with multiple enemies. I added a feature of speed which decreases when you use block or heavy attack. It affects the chance of a successful block and you will also die when it reaches 0. 
- there are 3 dungeons and a final boss
- there are checkpoints before each dungeon
- dungeons don't include their own boss

## Status
This project is a **work in progress** — more features are planned.

## How to run (Linux and macOS only)
To run this code you need to run these commands in your terminal:
chmod +x fantasygame.sh
./fantasygame.sh
- You need to run these commands in the **directory where the game is**, should be ~/Downloads/bash_game/ depending on your language.
